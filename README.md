Shri – College Enquiry Chatbot

1.📌 Project Overview

Shri is a college enquiry chatbot system developed as a 4-member team project. The system is designed to answer student queries related to college information using a chatbot interface.

This repository currently contains the frontend implementation and client-side chatbot logic. The backend services and database integration were implemented separately using Python Flask and MySQL.


---

2.🧱 Project Architecture

The project follows a modular architecture:

Frontend: HTML, CSS, JavaScript

Backend: Python Flask (API layer)

Database: MySQL


Frontend (UI + JS Logic)
        │
        ▼
Python Flask Backend (API)
        │
        ▼
     MySQL Database
